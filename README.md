# 🔧 Site em Manutenção

Uma página elegante e moderna para informar visitantes que o site está passando por melhorias.

## 📋 Sobre o Projeto

Este projeto é uma página de "Em Manutenção" desenvolvida com Next.js 14. Apresenta um design moderno e responsivo com animações suaves, ideal para ser utilizada durante períodos de atualização do sistema.

## 🚀 Tecnologias

- **Next.js 14** - Framework React com App Router
- **React 18** - Biblioteca para interfaces
- **CSS3** - Estilização com animações
- **Git** - Controle de versão
